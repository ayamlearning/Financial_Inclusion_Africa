# Financial_Inclusion_Africa
Can you predict who in Africa is most likely to have a bank account?

## Description of Problem
The objective of this model is to develop a machine learning model to predict which individuals are most likely to have or use a bank account. The model developed can provide an indication of the state of financial inclusion in Kenya, Rwanda, Tanzania and Uganda, while providing insights into some of the key factors driving individualsâ€™ financial security.

This problem was adapted from [Zindi](https://zindi.africa/competitions/financial-inclusion-in-africa)

## Running solution

- You can then run the docker container using port which is running on port **`4000`**
